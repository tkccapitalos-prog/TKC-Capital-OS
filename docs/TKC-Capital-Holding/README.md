# TKC Capital Holding

Projeto estratégico para criação da holding TKC Capital em Portugal.

## Objetivo

Criar uma estrutura empresarial para organizar, proteger e desenvolver os negócios do grupo:

- Fachada Figurada
- TKC Talent
- TKC Capital OS
- TKC Brasil Invest
- futuros projetos imobiliários, hotelaria, construção e IA

## Data prevista de constituição

Objetivo recomendado: 2027  
Preparação estratégica: 2026
